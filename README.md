# frontend-monthly

## local preview

```
npm ci
```

```
npm run dev
```

open `http://localhost:3000`

## add post

create md file into the `/src/content/posts` dir.

## add profile image

add image({userName}.jpg) file into the `/public/img` dir.

## publish

merge PR or push to `master` branch then starting auto publish by GitHub Actions.
